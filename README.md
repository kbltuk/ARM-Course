# Курс: Разработка АРМ системы на WPF и C#

## О курсе
Курс посвящен разработке Автоматизированного Рабочего Места (АРМ) на платформе .NET с использованием WPF и C#. В рамках курса мы создадим полноценное приложение для управления складскими операциями или продажами.

## Технологический стек
- .NET 6/8
- WPF
- Entity Framework Core
- MS SQL Server / PostgreSQL
- MVVM Pattern
